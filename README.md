# Developer Analytics Dashboard

Track and visualize club members' open-source contributions across GitHub: PRs, Issues, languages, repos, and leaderboard.

## Features
- Fetch contributions via GitHub API
- Aggregate per contributor and per repo
- Heatmaps, streaks, and leaderboard
- Export reports for events and placements

## Tech stack
- Backend: Node.js or Python scripts
- DB: Postgres or SQLite
- Frontend: React + charts

## Quickstart
1. `git clone <repo>`
2. Configure `GITHUB_TOKEN` in `.env`
3. `npm install` or `pip install -r requirements.txt`
4. Run data fetch script and start frontend
